# Sentinel Branding Kit

This repository contains all the essential Sentinel branding assets you may need.

## Repository Structure

```
Branding_Kit/
├── BK_PDF/     # PDF format assets (print-ready, high resolution)
├── BK_PNG/     # PNG format assets (web-friendly, transparent backgrounds)
└── BK_SVG/     # SVG format assets (scalable vector graphics)
```

## Brand Guidelines

When using these assets, please ensure:
- Maintain proper spacing and proportions
- Use approved color schemes
- Keep logos and designs unmodified unless specifically permitted
- Follow any licensing requirements
